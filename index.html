The Project

Easy!Appointments is a highly customizable web application that allows your customers to book appointments with you via the web. Moreover, it provides the ability to sync your data with Google Calendar so you can use them with other services. It is an open source project and you can download and install it even for commercial use. Easy!Appointments will run smoothly with your existing website, because it can be installed in a single folder of the server and of course, both sites can share the same database.



Features

Customer and Appointment Management

Easy!Appointments will give you access to all of your customers and their appointment history. You can manage your records easily using the administration section of the system.

Services and Service Providers Organization

You will be able to add the services and the service providers of your own company. Every provider will have a unique calendar in which he'll be able to view his appointment details.

Business Work Flow and Booking Rules Setup

The system can be configured to keep up with the operating rules of your business. Your customers will be able to book appointments only during the working hours.

Google Calendar Sync

Easy!Appointments supports appointment synchronization with Google Calendar so that you can use your data with other applications.

Email Notifications

Whether you are a customer or a service provider you will be notified of any action made on the system including booking, editing or canceling an appointment.

Stand Alone Installation

You can install your own copy on your server and access the code files and database information in any way you like. Easy!Appointments is open and free!

Translated User Interface

Easy!Appointments supports localization and is currently translated into English, German, French, Italian, Spanish and many more (see bottom list). More translations will be added in the future.

User Community

Whether you face problems or you want to add your own functionality Easy!Appointments has a large community of enthusiasts that will help you achieve what you want.

Installation & Update

Installation

Since Easy!Appointments is a web application, it runs on a web server and thus you will need to perform the following steps in order to install the system on your server:

  1. Make sure that your server has Apache, PHP and MySQL installed.
  2. Create a new database (or use an existing one).
  3. Copy the "easyappointments" source folder on your server.
  4. Set zour server information to the "config.php" file.
  5. Open your web browser and navigate to the installation directory URL.

That's it! You can now use Easy!Appointments at your will! Read the detailed installation guide on this wiki page.


Update

There are three things that you will need to do in order to update an existing Easy!Appointments installation.

  1. Backup your Easy!Appointments files and database.
  2. Copy and replace all the files from the new version except from "config.php".
  3. Open your browser to the URL "http://url-to-ea-installation/index.php/backend/update".

Read the detailed update guide on this wiki page.
