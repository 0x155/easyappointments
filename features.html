#Features

Customer and Appointment Management

Easy!Appointments will give you access to all of your customers and their appointment history. You can manage your records easily using the administration section of the system.

Services and Service Providers

You will be able to add the services and the service providers of your own company. Every provider will have a unique calendar in which he'll be able to view his appointment details.

Business Work Flow and Booking Rules

The system can be configured to keep up with the operating rules of your business. Your customers will be able to book appointments only during the working hours.

Google Calendar Sync

Easy!Appointments supports appointment synchronization with Google Calendar so that you can use your data with other applications.

Email Notifications

Whether you are a customer or a service provider you will be notified of any action made on the system including booking, editing or canceling an appointment.

Stand Alone Installation

You can install your own copy on your server and access the code files and database information in any way you like. Easy!Appointments is open and free!

Translated User Interface

Easy!Appointments supports localization and is currently translated into English, German, French, Italian, Spanish and many more (see bottom list). More translations will be added in the future.

User Community

Whether you face problems or you want to add your own functionality Easy!Appointments has a large community of enthusiasts that will help you achieve what you want.

Multiple Attendants

It is possible that some services accept more than one attendants for the same time slot. No worries, Easy!Appointments can do that too!

Dedicated Provider & Service Links

After setting the providers and services of your company, you will be able to get dedicated links for use in your homepage. These URLs can point to specific providers, services or both at the same time.

Responsive Design

The whole user interface is responsive! You can book or handle appointments from any viewport size without problems.

REST API

Use the flexible REST API endpoint and handle all the application-wide data from your clients. With carefully structured resources and available documentation you have the complete control.

#Translations

Easy!Appointments supports localization and many users have contributed to the project with their translations! The user interface is currently translated into English, German, Greek, Portuguese, Spanish, Dutch, Japanese, French, Chinese, Polish, Italian, Luxembourgish, Danish, Hungarian, Slovak, Russian, Finnish, Romanian, Turkish, Hindi and Bulgarian! You can add your own language or use one of the existing available on this page.
